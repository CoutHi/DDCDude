# Dependencies

## Fedora
```
gtk4-devel gcc cargo
```

## Debian 
```
libgtk-4-dev build-essential cargo
```

## Arch
```
gtk4 base-devel cargo
```

# How To Run
Clone this repository and run
```
cargo run
```
To run the program without compiling.

You can use
```
cargo build --release
```
To compile it.

The binary will be in target/release/ddcdude
